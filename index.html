Security-oriented projects I’m working on

File Encryption and Decryption

Security

Goal: protect files by converting readable data into a secure form and restoring it only when needed.

A tool that allows users to encrypt and decrypt files, keeping content safe from casual access. The main idea is simplicity: pick a file, apply encryption, and later decrypt it with the correct input so that sensitive data stays private.

Fake News Detector

Trust & Content

Goal: help users pause and think before sharing suspicious content.

A project that analyses given text (like headlines or short news snippets) and flags content that looks misleading or doubtful. The aim is to push people to double-check information instead of forwarding everything blindly.

Password Complexity Checker

Utility

Goal: make users aware of how weak or strong their passwords are.

A project that checks basic properties of a password—length, mix of characters, and simple patterns—and gives feedback on how strong it is. The idea is to motivate people to move away from easy-to-guess passwords to something more secure.
